# 🐘 PostgreSQL - Exercícios

Repositório contendo a resolução dos exercícios realizados durante o curso de **PostgreSQL** da plataforma **Udemy**.

O objetivo deste projeto é registrar minha evolução na linguagem SQL, praticando consultas, manipulação de dados e recursos do PostgreSQL.

---

## 📚 Conteúdo

Atualmente o repositório contém:

- Exercícios de Consultas Simples
- Scripts SQL utilizados na resolução dos exercícios

---

## 🛠️ Tecnologias utilizadas

- PostgreSQL
- SQL

---

## 🎯 Objetivos

- Praticar comandos SQL
- Aprender a manipular bancos de dados relacionais
- Desenvolver consultas cada vez mais complexas
- Consolidar conhecimentos em PostgreSQL
